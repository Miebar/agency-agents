Lista dostępnych komend NEXUS. Użyj konkretnej komendy z opisem zadania jako argumentem.

Dostępne komendy:

/nexus/full [opis projektu]
  — Pełny cykl produktu od zera (12-24 tygodnie, wszystkie agenty, Phase 0-6)

/nexus/sprint [opis feature/MVP]
  — Feature lub MVP (2-6 tygodni, 15-25 agentów, Phase 1-5)

/nexus/mvp [opis produktu]
  — Startup MVP z tygodniowym harmonogramem (4-6 tygodni)

/nexus/enterprise [opis funkcji]
  — Enterprise feature z compliance i canary deploymentem (6-12 tygodni)

/nexus/incident [opis incydentu]
  — Natychmiastowa reakcja na incydent produkcyjny (P0-P3)

/nexus/micro-bugfix [opis buga]
  — Szybka naprawa buga (Backend Architect + API Tester + Evidence Collector)

/nexus/micro-campaign [opis kampanii]
  — Kampania marketingowa (Social Media Strategist + team + Brand Guardian gate)

/nexus/micro-performance [zakres]
  — Diagnoza i optymalizacja wydajności

/nexus/micro-research [domena]
  — Research rynkowy + executive brief

/nexus/micro-ux [feature/produkt]
  — Badanie UX + projekt usprawnień + implementacja

/nexus/micro-compliance [zakres: GDPR/CCPA/ALL]
  — Audit compliance + raport dla stakeholderów
